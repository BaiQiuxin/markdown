    1. 创建分支可以通过

    ```git
    git branch <name>

    ```

    切换分支可通过

    ```git
    git switch <name>
    git checkout <name>
    ```

    删除分支可通过

    ```git
    git branch -d <name>
    ```

    当该分支的修改已经合并到其他分支时或者主分支，分支的修改不会丢失时，可以安全地删除该分支。

    ```git
    git reset HEAD <file>
    ```
